# Logitech-g-hub-mouse-gestures

[**English**](README.md)

MacOS系统下Logitech g hub上使用的,用来实现类似Master系列功能的鼠标手势Lua脚本.

本脚本借鉴了[g-hub-mouse-gestures],并在此基础上进行了修改,实现了罗技Master系列的窗口导航功能.
